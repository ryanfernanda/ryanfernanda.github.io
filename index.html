Indonesian Earthquake Data

Exploratory Data Analysis

A data-driven project that analyzes and visualizes earthquake events in and around Indonesia from 2000 to March 7, 2025, focusing on magnitudes of 3.0 and above. Using data from the USGS Earthquake Catalog, the dataset was segmented by time range, cleaned to remove duplicates and inconsistencies, and consolidated into a standardized format containing 59,570 records across 21 key parameters. The resulting dataset supports an interactive dashboard that enables users to explore spatial and temporal seismic patterns across the region.

Key Features:

  • USGS Web for earthquake data collection
  • Python (Pandas & NumPy) for data cleaning
  • Tableau for interactive data visualization

Shallow Foundation Analysis

Machine Learning

A study to analyze the safety factor (SF) of shallow foundations against earthquakes using Kolmogorov-Arnold Networks (KAN). The dataset consists of 3,840 samples with six key parameters, including peak ground acceleration (PGA), foundation width (B), depth (DF), soil density (Y), and structural load (P). KAN was applied and compared with several other algorithms, including MLR, SVR, KNN, MLP, and TabNet, showing superior performance based on RMSE, MAPE, R-Squared, and A20 Index.

Technical Details:

  • Python for computing and data exploration
  • KAN model and its comparison
  • Google Colab for cloud-based model training

Shallow Foundation Analysis

Web Development

After developing the Kolmogorov-Arnold Networks (KAN) model to predict safety factors (SF) for shallow foundations, I deployed it into an interactive web platform using Python Flask. The platform allows users to input necessary values, along with visual representations of foundations. The system categorizes the SF result into three classifications: Safe, Not Safe, and Over Safe. Additionally, it stores input-output data dynamically, enabling users to download results as a CSV for further analysis.

Technical Details:

  • Python Flask for web development
  • KAN model integration for SF prediction
  • Dynamic data storage with CSV export

Bearing Capacity Analysis I

Machine Learning

A comparative study to evaluate various regression algorithms for predicting ultimate bearing capacity (Q_ult) of strip foundations on sandy clay slopes. The dataset consists of 1,439 samples with seven features, including slope angle (β), foundation width (B), depth (Df), internal friction angle (φ), soil unit weight (γ), and vertical load (P). Multiple regression models were tested with default parameters, including another ensemble and DL models. XGBoost performed the best based on evaluation metrics.

Key Features:

  • Python for computing and data exploration
  • Regression models comparison (MLR, etc.)
  • Performance evaluation with default params
